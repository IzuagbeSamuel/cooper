Best Visual Studio Code Extensions

June 10, 2021/ By Samuel Izuagbe

Making web development very interesting and easy for you to use. I have compiled a series of good VSCode Extensions to help easy and make your codes neat, faster to code and more indented. VSCode Extensions let you add languages, debuggers, snippets, formatter, and so on...

Here are list of VScode Extensions:

Live Server

I know this is the main one you guys must be craving for because this extension has really helped a lot of Web Developers. This extension helps to run the web page continuously without refreshing the page.

Follow this few steps to install Live Server on your Visual Studio Code

  1. Open your VScode on your computer, if you have not yet installed then you can do that Here
    2. .

    After Installing, open the VScode editor.

  2. Create a basic file in the VScode editor.

    3. Create an index.html file

  3. Navigate to Extension at the left side navigation menu and click on it
  4. Search on the search bar at the top - Live Server
  5. Click on install to add the extension to your editor
  6. Once installed, go to your index.html positioned file on the editor; right click and click open with live server or you can use the keyboard shortcut Alt+L Alt+O. The web page will open on your default browser with a port 5500


Auto Rename

You need to change names to tags on your codes, Auto Rename got you covered. It changes the tag name for you easy and with no stress. Change the start tag, the end tag changes automatically. How cool is that right? It helps us code fast with less bugs on your codes

Here's how to install Auto Rename Extension

  1. Open your VScode editor
  2. Navigate to the Extension on the side navigation menu and click on it
  3. Search for Auto Rename and install it

    4. Boom! you are ready to go!
